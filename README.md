# Number-guessing-game-

A fun and interactive **Python-based number guessing game** where the player tries to guess a randomly generated number within limited attempts.

##Features

-  Multiple difficulty levels:
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–500)
-  Limited attempts based on difficulty
-  Score system based on performance
- Play again option
-  Random number generation using Python


## Tech Stack

- **Language:** Python  
- **Module Used:** random
